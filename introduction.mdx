# Welcome to Frappe HR

Frappe HR is a complete, open-source Human Resource Management System (HRMS) built on the [Frappe Framework](https://frappeframework.com). It covers the full employee lifecycle — from onboarding to exit — and integrates seamlessly with [ERPNext](https://erpnext.com) for accounting, payroll payments, and more.

## What is Frappe HR?

Frappe HR is a full-featured HRMS with over 13 modules that handles everything from hiring to retirement. It started as a set of modules within ERPNext and became a standalone product from version 14 onwards.

## Key highlights

- **Open source** — GPL-3.0 licensed, self-hostable, no vendor lock-in
- **Mobile-first** — Progressive Web App (PWA) for leaves, attendance, and salary slips
- **ERPNext integrated** — Seamless accounting, payments, and project integration
- **Regional support** — India-specific income tax, HRA exemption, and gratuity calculations
- **Automated workflows** — Multi-level approvals, reminders, and scheduled tasks
- **13+ modules** — Everything from onboarding to exit interviews in one platform

## Architecture

Frappe HR is a Frappe Framework application:

- **Backend**: Python, using Frappe's ORM and REST API
- **Frontend (Desk)**: JavaScript with Frappe's form and list views
- **Mobile App**: Vue 3 + Ionic (PWA), bundled separately in `frontend/`
- **Database**: MariaDB (via Frappe bench)

<Note>
Frappe HR requires both the **Frappe Framework** (≥17.0) and **ERPNext** (≥17.0) to be installed on the same bench site.
</Note>
